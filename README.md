 Student Performance Tracker

A simple CRUD-based web application using PHP, MySQL, HTML, CSS, and JavaScript.

 Features
- Add student information
- View existing student records
- Edit student grades
- Delete student records
- Uses relational database tables

 Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP

 Database Structure
- students
- subjects
- grades

 How to Run
1. Import the database in phpMyAdmin
2. Place project folder in htdocs
3. Run Apache and MySQL (Must have XAMPP installed)
4. Open `localhost/student_performance/index.php` in browser hehe
